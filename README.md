# Quality Muffler Inventory System

A full inventory management system built for a local auto shop in Visalia, CA.

## Features
- Add, edit, and delete inventory parts
- Restock and deduct quantities with one click
- Low-stock email alerts at quantity 2 (warning) and 1 (urgent) via EmailJS
- Filter view for low stock items
- Print-ready inventory sheet
- PostgreSQL database with 200+ parts via Supabase

## Stack
HTML · Tailwind CSS · Supabase (PostgreSQL) · EmailJS · Netlify
